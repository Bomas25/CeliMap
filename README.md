# CeliMap
CeliMap is a web-based application designed to help users find gluten-free restaurants, cafés and bakeries in their local area. The application is aimed at people with coeliac disease or gluten intolerance and provides a clear, map-based overview of suitable establishments.

# Project description
The application displays local businesses from an SQLite database on an interactive map.
The user’s current location can be determined via browser geolocation.
In addition, new local businesses can be added to the database via a form.

An optional route function allows the route from the current location to the selected destination to be displayed.

 # Features
- Display of the user’s location via browser geolocation
- Interactive map view based on OpenStreetMap (Folium)
- Display of restaurants, cafés and bakeries from an SQLite database
- Filtering by radius and category
- Adding new entries via a web form
- Route calculation to a selected venue
- Automated unit tests for quality assurance
